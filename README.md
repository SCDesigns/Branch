# Branch

- A Sinatra CMS application for local events & meetups (Early Stage)

## Installation / Usage
    
	1. Clone the repository 
	2. Run `rake db:migrate`
	3. Run `shotgun`
	4. Open the given address in your browser and continue as you wish
